# 08. Vendor risk и независимость от платформы

> **Контекст для AI-агента**: SimpleWine Studio использует OpenRouter (US) → Anthropic Claude Opus 4.6/4.7 (US). Anthropic не работает с РФ-юрлицами напрямую. Этот блок — про риски привязки к зарубежному вендору и план митигации. Полный контекст в [`00_CONTEXT.md`](./00_CONTEXT.md). Шкала: 🟢 / 🟡 / 🔴 / ⚪. Сводный итог в [`09_FINAL_VERDICT.md`](./09_FINAL_VERDICT.md).

## Принцип блока

Меряем не «надёжен ли поставщик», а **«что произойдёт, если поставщик завтра отключит РФ / поднимет цены 2× / закроет API»**.

## Реальные риски (признаём)

| Риск | Вероятность | Импакт |
|---|---|---|
| OpenRouter блокирует РФ | средняя | пайплайн встанет на часы |
| Anthropic блокирует OpenRouter | низкая | переключение на Claude через альтернативный посредник (1 день) |
| Anthropic поднимает цены 2× | средняя за год | бюджет токенов растёт ~50% (т.к. часть токенов уже была дорогой) |
| Курс USD/RUB +30% | высокая | пропорциональный рост в рублях |
| Sanctions update (РФ vs US) | средняя | может потребоваться смена юрлица |
| Anthropic закрывает Opus | низкая | переход на Claude Sonnet/Haiku или другую модель |

## Митигации (что уже заложено в архитектуру)

### 1. Multi-model готовность

LLM-провайдер меняется **одной строкой** в коде (endpoint + ключ). Тестировали:
- ✅ Claude Sonnet — работает, креативность чуть ниже, цена 5× меньше
- ✅ Gemini Pro — работает, отличается tone (нужна доподстройка промптов)
- ✅ Yandex GPT — работает с заметной просадкой по премиум-tone, но допустимо для простых форматов
- ⚠️ Self-hosted Qwen 3 / DeepSeek V3 на Yandex Cloud GPU — план для on-prem-чувствительных шагов

### 2. Иностранное юрлицо для договора

Стандартная практика для работы с OpenAI/Anthropic из РФ. Не серая зона — есть юристы, которые этим занимаются. **Часть подготовки пилота.**

### 3. Fallback-стратегия

```
[Бриф] 
   ↓
[Pre-processing на YaGPT/GigaChat] ← PII-маскирование (РФ)
   ↓
[Generation на Opus / fallback Sonnet / fallback YaGPT]
   ↓
[Validator на детерминированном JS] ← всегда работает, не зависит от LLM
   ↓
[Output]
```

Валидатор и обвязка **полностью независимы от LLM**. Если завтра отрубят Anthropic — мы потеряем 30% качества текста, но **не потеряем compliance, не потеряем процесс, не потеряем интеграции**.

### 4. Self-hosted backup для критичных задач

Yandex Cloud GPU + Qwen 3 / DeepSeek V3 — реалистичный self-hosted вариант для шагов с PII или для on-prem-требований IT.

## Вердикт по конкурентам

- **ChatGPT/Claude любой SKU**: 🟢 — для пользователя там модель = весь продукт. Если OpenAI блокирует — Custom GPT превращается в тыкву. У нас — обвязка переживает.
- **ChatGPT Enterprise**: 🟡 — частично решается через Azure OpenAI (Microsoft hosting), но всё равно один провайдер модели.
- **AlpinaGPT**: 🟡 — у них как раз multi-model внутри (это их main feature). Но обвязки нет, валидатора нет, интеграций нет — то есть антифрагильность только на уровне «где запросить токен», не на уровне продукта.
- **Yandex GPT / GigaChat**: смешанный — у нас перед ними преимущество, что мы можем включить их же модель в наш пайплайн как fallback. У них — только их модель.

## Готовый чеклист для IT-директора (по vendor risk)

| Вопрос IT | Ответ |
|---|---|
| Что если Anthropic заблокирует РФ? | Switch на Sonnet/Gemini/YaGPT, 1 день, потеря ~30% качества на креативе |
| Что если OpenRouter упадёт? | Прямой Anthropic API через альтернативного посредника, или альтернативная модель |
| Кто оплачивает токены? | Иностранное юрлицо (часть пилота), счета в рублях для SimpleWine |
| Что если санкции против Anthropic? | Self-hosted Qwen/DeepSeek на Yandex Cloud (Q3 roadmap), весь обвяз остаётся |
| Можем ли локализовать всё в РФ? | Да — на YaGPT с просадкой качества или Qwen self-hosted с инвестицией в GPU |
| Что мы НЕ можем? | Получить on-prem самой Opus — Anthropic не отдаёт веса. Это ограничение **рынка**, не наше |

## Главный аргумент для презентации

> «У ChatGPT и Claude **модель = весь продукт**: блокируют OpenAI — Custom GPT становится бесполезным. У нас **модель = одна из компонент**, обвязка работает на любой LLM. Это не теория — мы тестировали на Sonnet, Gemini, YaGPT. Поэтому даже худший сценарий — отключение Anthropic — для нас означает 30% потери качества креатива, не остановку процесса. **Наш моат — обвязка, а не модель.**»
