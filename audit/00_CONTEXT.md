## 2. Реальная архитектура (по коду, не по маркетингу)

- **Frontend**: vanilla JS SPA, никакого фреймворка, статические файлы
- **Backend**: НЕТ. 100% работает в браузере
- **Деплой**: GitHub Pages (`docs/index.html`) или локально `npm run serve`
- **LLM-провайдер**: OpenRouter API → **Anthropic Claude Opus 4.6 / 4.7** (НЕ Sonnet — это критично для расчётов цены)
- **Альтернативные модели**: через OpenRouter одной строкой кода
- **Хранение настроек**: `localStorage.briefParser_settings` (включая API-ключ в plain-text)
- **Зависимости**: mammoth.js (DOCX→HTML), turndown (HTML→MD), reveal.js 5.1.0

### Что система реально делает (не маркетинговое):

- **НЕ** "15-17 параллельных агентов" (это маркетинговая формулировка). Это **последовательный single-LLM с repair loop** (max 3 попытки), где разные шаги используют разные системные промпты ("писатель", "редактор" — это разные роли одной модели).
- **Что есть на самом деле**:
  - **BriefParser** (regex-based для multiformat, LLM-based для email с PARSING_SPEC)
  - **PromptComposer** — собирает OUTPUT_CONTRACT для 13 форматов
  - **Generator** — один вызов Claude Opus 4.6/4.7 с temperature 0.3
  - **ValidatorMultiformat** — детерминированный JS-валидатор (лимиты, чёрные списки)
  - **SurgicalRepair** — точечные исправления полей (subject/preheader/body/cta), при кросс-полевых нарушениях fallback на полный регенерат
  - **TieredKnowledge** — оптимизация токенов (150-2500 vs 6300 full knowledge)

### 4-уровневый каскад знаний (всё в `knowledge/` как .md файлы):

1. `prompts/system_instructions.txt`

## 5. Цены LLM (для блока Cost)

**Anthropic Claude Opus 4 / 4.6 / 4.7** через OpenRouter:
- Input: **$15 / 1M токенов**
- Output: **$75 / 1M токенов**

**Это самая дорогая модель на рынке** — выбрана осознанно ради качества compliance-валидации в критичном домене (реклама алкоголя).

Для оптимизации можно использовать routing:
- Opus 4.6/4.7 — основная генерация
- Haiku / Gemini Flash — подсчёт символов, парсинг структуры (в 5-10× дешевле)
- YaGPT / GigaChat — fallback и российские задачи

## 6. Реальные unit-economics (для блока Cost)

### Multiformat-прогон (1 бриф → 65 текстов):
- Input: ~7800 × 1.3 (с repair loop) = 10 000 токенов → **$0.15**
- Output: ~4000 × 1.3 = 5200 токенов → **$0.39**
- **Итого: ~$0.54 / ~50 ₽ за 65 текстов**

- Input: ~3000 + Output: ~3500 + repair ×1.2 → **~$0.37 / ~34 ₽**
- 200 статей в месяц = **~6 800 ₽/мес токенов**

### Баннер:
- ~1000 in + 200 out → **~$0.03 / ~3 ₽**

### Месячный бюджет 1 копирайтера (с 30 доработками вместо 5):
**~15-25 тыс ₽/мес токенов**

### 3 копирайтера:
**~50-75 тыс ₽/мес токенов**

### Декомпозиция 220 000 ₽/мес:

## 7. Honest gaps (то, что Вера ДОЛЖНА сама озвучить раньше IT)

1. **API-ключ в plain-text localStorage** → план фикса: серверный прокси + Vault
2. **Нет audit-log** → план фикса: append-only лог в Yandex Object Storage
3. **Нет PII-маскирования брифа** → план фикса: pre-processing «редактор» (Алина сама про него говорила)
4. **Vendor lock-in** на OpenRouter+Anthropic → митигация: иностранное юрлицо, fallback на YaGPT, self-hosted Qwen
5. **Bus factor = 2** → план фикса: RUNBOOK + golden tests + branch protection
6. **30 доработок vs 5** — рабочий стиль, не лимит → митигация: template-бриф + чеклист
7. **Отсутствие true on-prem для Opus** → митигация: self-hosted Qwen/DeepSeek на Yandex Cloud GPU

Все 7 — это **roadmap пилота**, не недостатки продукта. Подача: «вот что слабо, вот срок фикса, вот что входит в стоимость пилота».

## 8. Юрисдикция и риски

- **Anthropic не работает с РФ-юрлицами** — доступ через OpenRouter (US/Canada посредник)
- **Оплата в USD** через карты/Wise/иностранные ИП — для корп-договора это серая зона
- **Если OpenRouter заблокирует РФ** — пайплайн встанет до переключения на другую модель (часы работы)
- **Курс USD↔RUB** — токены в долларах, валютный риск

## 9. Главные нарративные ходы

1. **«Почтовый клиент vs почтовый сервер»** — переформулировка категории. AlpinaGPT/ChatGPT/Claude — окна для разговора с моделью. SimpleWine Studio — production runtime для контента, который случайно использует LLM.
2. **«Знания — не моат, валидатор — моат»** — лексикон любой может скопировать в Custom GPT за неделю; что нельзя — это детерминированный валидатор + repair loop.
3. **«Антифрагильность через multi-model»** — наша обвязка переживает смену LLM; у конкурентов модель = весь продукт.
4. **«Окно vs Vinlab × Yandex»** — у них минимум 6 месяцев на повторение валидатора и обвязки; решение нужно сейчас, не через квартал.
5. **«ROI не в токенах, а в фонде оплаты труда»** — экономия 800к/мес vs пилот 220к.
