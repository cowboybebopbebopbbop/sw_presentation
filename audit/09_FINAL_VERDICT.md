## 5 главных моатов SimpleWine Studio

1. **Детерминированный валидатор поверх LLM** — не воспроизводится промптом, защищает от ФАС категорически
2. **Multiformat-консистентность** — 65 текстов одним прогоном с гарантией согласованности % скидки
3. **Знания и процесс в git** — bus factor, version control, code review, golden tests
4. **Архитектура production runtime** — DOCX-импорт, HTML-вывод, Gemini для картинок, готовность к Mindbox/CMS/PIM
5. **Антифрагильность** — обвязка переживает смену LLM (Sonnet/Gemini/YaGPT/self-hosted Qwen)

## 5 честных слабостей (выложить ПЕРВЫМИ)

1. **Голая входная цена дороже** любой подписки (199₽–60$/user)
2. **API-ключ в plain-text localStorage** → план фикса в 2 нед пилота (серверный прокси + Vault)
3. **Нет audit log сейчас** → 3 нед пилота (Yandex Object Storage + PDF экспорт)
4. **US-юрисдикция OpenRouter+Anthropic** → митигация через иностранное юрлицо + multi-model fallback на YaGPT
5. **Bus factor 2** сейчас → 3+ через RUNBOOK + golden tests (часть пилота)

## 8 готовых talking points для встречи (в порядке использования)

### 1. Открывающий — категория продукта
> «Мы — не ещё один помощник для копирайтера. Мы — production runtime для контента, который случайно использует LLM. Это разница между почтовым клиентом и почтовым сервером.»

### 2. Ответ на «обычная нейронка»
> «Голый ChatGPT — это писатель. Custom GPT — писатель + знания. У нас — писатель + редактор + юрист. Юрист — детерминированный код, его нельзя обмануть галлюцинацией модели. Один штраф ФАС = 6 месяцев нашего пилота.»

### 3. Ответ на «AlpinaGPT за 2к/13к»
> «AlpinaGPT — это маршрутизатор к ChatGPT/Claude/Gemini за рубли. Это ChatGPT в кириллице. Сравнивать с нашим пилотом всё равно что сравнивать Word с CMS. Можно, но печатать счета-фактуры в Word вы не будете. AlpinaGPT не имеет валидатора, не имеет git, не интегрируется с Mindbox.»

### 4. Ответ на «дорого 220к»
> «Декомпозиция: ~60к токены, ~80к платформа, ~80к доработки и интеграции. Это услуги, не подписка на чат. Под низкий объём есть сценарий A — только SEO за ~67к. Под оптимизацию — сценарий D с multi-model routing за ~170к.»

### 5. Ответ на ROI
> «Статья: было 7 часов копирайтера, стало 2.5. На 200 статьях экономия фонда оплаты труда — около 800 тысяч в месяц. Пилот окупается в 3.5–5.5×, точка безубыточности на 50 статьях.»

### 6. Ответ на Vinlab × Yandex
> «Vinlab подписал партнёрство с Яндексом, но публичных результатов нет. У нас валидатор, golden tests, RUNBOOK, ребрендинг лексикона PR-ом за 2 минуты — мы это делаем сейчас, не через квартал. Окно для решения — сейчас.»

### 7. Ответ на «безопасность»
> «У нас сейчас 5 честных слабостей по security — мы их знаем и приносим план фикса как часть пилота. У ChatGPT/Claude в чате — те же слабости плюс модель учится на ваших данных. ChatGPT Enterprise лучше нас по аудиту, но без нашего валидатора и за тот же бюджет на seats.»

### 8. Ответ на «vendor lock-in»
> «У ChatGPT/Claude модель = весь продукт. У нас модель = одна из компонент. Тестировали Sonnet, Gemini, Yandex GPT — обвязка работает на любой. Антифрагильность — наш моат.»
